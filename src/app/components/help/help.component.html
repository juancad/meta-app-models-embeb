El objetivo principal de este trabajo es ofrecer un entorno mediante el cual, una persona que cuente con un sistema de Inteligencia Artificial para la clasificación de imágenes pueda generar de forma sencilla un entorno web que le permita poner en producción modelos mediante el uso de los dispositivos necesarios, tales como la cámara web de un smarthphone, PC, tablet... Esta cámara podrá capturar una imagen en tiempo real y, haciendo uso de un Sistema Inteligente basado en visión por computador previamente optimizado, ́este podrá dar respuesta a lo que la cámara “está viendo” en cada momento.

Cualquier modelo de Keras para la clasificación de imágenes, transformado al formato de capas de TensorFlow.js es compatible con la aplicacíon. Para saber cómo realizar esta transformación, visitar el siguiente enlace: https://www.tensorflow.org/js/tutorials/conversion/import_keras?hl=es-419

Para subir el modelo deberás pulsar en el botón "Crear nueva aplicacíon" de la página de inicio. Una vez pulsado aparecerá el formulario para crear una nueva aplicación. La aplicación deberá tener un identificador único para el usuario que la crea con un máximo de 20 caracteres y que sólo puede contener caracteres alfanuméricos, punto, guión y guión bajo. Una vez introducido el identificador se deberá seleccionar el modelo "model.json" y los archivos ".bin" generados con el método anteriormente descrito. Estos archivos no deberán ser modificados por el usuario (tampoco se les debe cambiar el nombre). En caso de ser modificados no se podrán subir. En el primer subidor de archivos se selecciona el modelo "model.json", y en el siguiente todos los archivos .bin (si no seleccionas todos los que debe tener el modelo tampoco se podrán subir).

Tienes que tener Python instalado en tu equipo. Desde un terminal situarte en el directorio de la aplicación descomprimido y donde se encuentre el index.html y ejecutar el comando:
python -m http.server 8000
Posteriormente acceder a la ruta del navegador: http://localhost:8000/
